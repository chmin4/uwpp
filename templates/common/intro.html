UWPP: Universal Web Project Platform

2024년에 진행했던 백엔드 프로젝트를 정리할 겸, 편하게 메모하거나 실시간 사진 첨부가 가능한 lecture note 프로그램이 마땅치 않아 이 사이트를 만들기 시작했다.

Universal Web Project Platform(범용 웹 프로젝트 플랫폼)이라는 이름은, 웹 서비스 프로젝트 실습을 위한 기반 플랫폼이라는 의미를 담고 있으며, 이는 사이트 개설 목적을 잘 나타낸다.

백엔드 프로젝트 정리에 중점을 두었기 때문에, 서버는 Raspberry Pi에 직접 올려 운영하고 있다.

사이트의 구성은 다음과 같다:
  • Memo Board – 간편한 메모장이자 강의 노트. 컴퓨터로 메모할 수 있는 나만의 공간을 찾기 어려웠다. 물론 일반 메모장도 있지만 로컬에 저장되다 보니 언제 어디서든 접근하기엔 불편했고, OneDrive와 연동되는 메모 프로그램은 종종 동기화 문제가 있어 만족스럽지 않았다. Google Keep도 써봤지만 강의 노트로 쓰기엔 적절하지 않았고, Notion은 메모용으로는 오히려 불편하게 느껴졌다. 그래서 직접 만든 기능이다.
  • Blog – 추후 추가 예정
  • Daily Platform – 학교 포털 사이트에서 착안한 일정 관리 플랫폼. 장기 계획부터 당장의 할 일까지 구성하고 실천할 수 있도록 도와주는 기능으로, 추후 스마트폰 앱으로도 확장할 예정이다. 현재는 기획 단계이며, 실제 개발은 제대 이후가 될 것으로 보인다.

혼자 사용할 목적이긴 하지만, Raspberry Pi 3b+ 기반 서버의 최대 동시 접속 가능 인원이 약 20명인 만큼 사용자별 sandbox처럼 동작할 수 있도록 로그인 기능을 구현했다. 회원 가입은 관리자가 직접 등록하는 방식이다.
